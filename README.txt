Break Me Completely — это прототип 2D-аркада, объединяющий механики из Angry Birds
и нарративную обертку легендарного мема Ломай меня полностью. Проект разработан
на Unity 6.

- Геймплейное ядро: Реализован полный игровой цикл. Настроена физика полета снарядов,
система нанесения урона и разрушения структур с учетом свойств материалов.

- Системная архитектура: Интегрированы современные системы Unity (Input System,
Universal Render Pipeline и тд).

- Визуал и UI: Сформирована целостная арт-стилистика. Настроен URP для 2D
(спрайты, пост-эффекты). Сверстаны главное меню и экран настроек.

- Масштабируемость: Архитектура построена по data-driven принципу. Добавление новых материалов,
блоков, снарядов и уровней не требует изменения кода, а осуществляется через создание
новых ассетов (ScriptableObjects).
